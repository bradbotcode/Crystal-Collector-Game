Fruit Collector Game

  • The game begins by displaying a random number in the green box below.
  • Each fruit has a mystery value set to it between 1 and 12.
  • When a fruit is clicked, it will add points to the box below. You can only determine the value of the fruit by clicking it.
  • To win, your total points must match the random number. If you go over, you lose.
